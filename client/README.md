# AutoService — Frontend

React 19 + TypeScript + Vite client for the AutoService Platform.

## Stack

- React 19 + TypeScript
- React Router v7
- SCSS Modules (BEM methodology)
- Tailwind CSS (utility classes)
- React Hook Form + Zod
- Custom fetch wrapper with automatic JWT refresh

## Development

```bash
npm install
npm run dev       # http://localhost:5173
npm run build
npm run preview
```

Requires the backend running on `http://localhost:5000`.
See the [root README](../README.md) for full setup instructions.
